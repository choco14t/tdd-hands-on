# TDD ハンズオン

[テスト駆動開発(TDD)オンライン勉強会 #1 「ライブコーディングで体感する TDD 基礎」 - connpass](https://ddd-community-jp.connpass.com/event/174617/)でライブコーディングされた内容を PHP でリライトしたものです。

## 実行環境

- PHP 7.3 以上

## セットアップ

```shell script
git clone git@github.com:choco14t/tdd-hands-on.git
cd tdd-hands-on
composer install
```

## テスト実行

```shell script
composer test
```
